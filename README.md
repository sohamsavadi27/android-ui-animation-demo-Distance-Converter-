#  Android UI Animation Demo

An Android application demonstrating basic UI animations and an exit confirmation dialog using Java and XML.

##  Features
1)Smooth UI animations (e.g., fade, scale, or translate)
2)Exit confirmation dialog (Are you sure you want to exit?)
3)Clean and simple user interface
4) Beginner-friendly Android implementation
---
## Tech Stack
1)Language: Java
2)UI:XML (Android Layouts)
3)IDE: Android Studio
---
## Project Structure
* `MainActivity.java` → Handles UI interactions and animations
* `activity_main.xml` → Defines layout and UI components
